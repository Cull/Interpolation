Данная программа для заданного точечного шаблона функции высчитывает её интерполирование значение:
На вход принимает два файла:
первый содержит начальные значения интеполируемой фукнции в формате:
n
x_1 f(x_1)
.
.
.
x_n f(x_n)

, где n- число точек интерполяции x_i и f(x_i) значение аргумента и функции для данного аргумента,
причем x_i упорядочены по возрастанию (пример set.txt).

второй файл содержит значения аргументов, для которых нужно вычислить значение функции, в формате:

n
x_1 
.
.
.
x_n 

, где число аргументов для вычиления значения функции (пример args.txt)

Так как gmail запрещает отправку exe файлов пришлось переминовать приложение  в interpol.bmp, 
т.е. его надо переименовать в *.exe.
Компилировал в IDE CodeBlocks компилятор gnu gcc c++11 iso, windows 10.
Входные файлы должны находиться в директории с exe файлом, выходной файл result.txt.
